# IEB Exam Preparation Dashboard

A polished, static study dashboard for Information Economics & Business (IEB) exam preparation.

This project contains a responsive HTML dashboard and four focused session pages that summarize core digital business concepts:

- **Session 1:** Digitalization, AI & Data
- **Session 2:** Network Effects & Digital Markets
- **Session 3:** Pricing & Revenue Models
- **Session 4:** Dynamic Capabilities & Organizational Agility

## Project structure

- `index.html` — main dashboard homepage with session navigation
- `session_01.html` — session 1 overview and key ideas
- `session_02.html` — session 2 overview and key ideas
- `session_03.html` — session 3 overview and key ideas
- `session_04.html` — session 4 overview and key ideas
- `session 03/`, `session 04/` — additional session-related assets or backup files (if present)

## Features

- clean, modern study interface built with HTML, CSS, and Tailwind utility styling
- session cards with core themes, strategic insights, and concept summaries
- fast local access without any build tools or server runtime

## How to use

1. Open `index.html` in a web browser.
2. Click any session card to view that topic page.
3. Use the "Back to Dashboard" button on each session page to return to the main dashboard.

## Notes

- The project is static, so no installation or package dependencies are required.
- Best used as a study reference for exam revision in digital strategy, platforms, pricing, and organizational agility.
- If you want to extend the dashboard, add new session pages and update `index.html` with new session cards.
